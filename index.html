Proyecto MIRROR · Guía
Armá tu MIRROR en 3 pasos

Tu MIRROR necesita tres cosas tuyas, y las tres son gratis. No hace falta saber programar: copiás, pegás y listo. Seguilas en orden. Al terminar volvés al constructor y pegás las tres direcciones.

1. Tu clave de IA · gratis (Groq)
  1. Entrá a console.groq.com y creá tu cuenta (podés entrar con tu Google).
  2. En el menú, andá a API Keys → Create API Key → ponele un nombre → Copiá la clave (empieza con gsk_...). Guardala: la vas a usar en el Paso 2.
⚠ Esa clave es secreta. NO la pegues en MIRROR. Va dentro de tu proxy (Paso 2), donde queda blindada.
2. Tu proxy · la caja fuerte (Cloudflare)

El proxy guarda tu clave del lado del servidor, para que nunca viaje al navegador. Es un "Worker" de Cloudflare, gratis.

⚡ La forma rápida — 2 clics

Tocá el botón: Cloudflare clona el proxy en tu cuenta, te pide tu clave de Groq (la del Paso 1) y lo deja andando. La dirección .workers.dev que te queda es tu PROXY. No tenés que copiar ni pegar nada de código.

▸ Deploy to Cloudflare
— o a mano, si preferís —
  1. Entrá a dash.cloudflare.com y creá una cuenta gratis.
  2. Workers & Pages → Create → Create Worker → ponele un nombre (ej: mi-mirror-api) → Deploy.
  3. Tocá Edit code: borrá todo lo que haya y pegá el código de abajo. Después Deploy.
  4. Settings → Variables and Secrets → Add → nombre GROQ_KEY, valor = la clave del Paso 1 → guardá y Deploy.
  5. Copiá la dirección que termina en .workers.dev. Esa es tu PROXY.

📧 Tu MIRROR responde tus mails

Tu MIRROR lee los mails que vos elijas y te deja borradores de respuesta escritos con tu estilo, en la carpeta Borradores de Gmail. Nunca envía solo: vos leés, corregís y mandás. Honesto por diseño.

✓ Gratis y sin clave nueva. Reusa el mismo proxy y la misma clave de IA que ya configuraste para tu MIRROR. Corre en los servidores de Google, dentro de TU cuenta. Sin servidor central, sin costo extra.
🖐 Vos tenés el control. MIRROR no escanea tu bandeja. Solo redacta los mails que VOS marcás a mano con la etiqueta MIRROR-contestar. El resto ni los mira.
Paso a paso
  1. Tocá Copiar mi script ya configurado (abajo): viene con tu proxy y tu estilo puestos.
  2. Entrá a script.google.com → Nuevo proyecto. Borrá todo y pegá el script. Guardá (💾).
  3. Ejecutá una vez mirrorCrearEtiquetas → crea en Gmail las etiquetas del flujo. (Google pide permiso para tus mails: aceptá, es tu cuenta.)
  4. En Gmail, ponele la etiqueta MIRROR-contestar a los mails que quieras contestar (🏷 → tildás la etiqueta). Un clic por mail.
  5. Probá seguro: ejecutá mirrorProbar → te muestra el borrador sin crearlo. Cuando te guste, ejecutá mirrorRevisarMails y revisá el borrador en Gmail.
  6. Opcional automático: mirrorInstalarDisparador revisa cada 15 min, pero siempre solo los mails que etiquetaste. Para frenar: mirrorQuitarDisparador.

// ─────────────────────────── CONFIG (completá esto) ───────────────────────────
const CONFIG = {

  // Tu proxy (el mismo .workers.dev de tu MIRROR) + la ruta del motor gratis.
  PROXY_URL: 'https://TU-PROXY.fisica-dat.workers.dev',
  RUTA:      'groq',

  // El modelo (para Groq gratis suele andar bien):
  MODELO:    'llama-3.3-70b-versatile',

  // IDIOMA de las respuestas (igual respeta el idioma del mail entrante).
  IDIOMA:    'español rioplatense',

  // TU ESTILO / QUIÉN SOS → para que el borrador suene como vos.
  // Pegá acá el mismo "núcleo" que pusiste en tu MIRROR.
  PERFIL: 'Describí acá quién sos y tu estilo de escritura (lo mismo que pusiste en tu MIRROR).',

  // ── CONTROL MANUAL: las dos etiquetas del flujo ──
  // La que VOS aplicás a mano a los mails que querés contestar:
  ETIQUETA_PEDIR: 'MIRROR-contestar',
  // La que el script pone cuando ya dejó el borrador (para no repetir):
  ETIQUETA_LISTO: 'MIRROR-borrador-listo',

  // Firma del borrador (dejala '' si no querés firma).
  FIRMA: '— [Tu nombre]',

  // Cuántos mails procesar por corrida.
  MAX_HILOS: 10,

  // Parámetros del modelo.
  TEMPERATURA: 0.5,
  MAX_TOKENS:  700,

  // Tope de caracteres del mail entrante que se manda al modelo (ahorra tokens).
  MAX_CHARS_ENTRANTE: 3500
};
// ───────────────────────────────────────────────────────────────────────────────


/** Crea en Gmail las dos etiquetas del flujo. Ejecutar UNA vez al principio. */
function mirrorCrearEtiquetas() {
  obtenerEtiqueta_(CONFIG.ETIQUETA_PEDIR);
  obtenerEtiqueta_(CONFIG.ETIQUETA_LISTO);
  Logger.log('✅ Etiquetas listas. En Gmail, ponele «' + CONFIG.ETIQUETA_PEDIR +
             '» a los mails que quieras que MIRROR conteste.');
}

/** PRUEBA SEGURA: arma el borrador del primer mail etiquetado, pero NO lo crea. */
function mirrorProbar() {
  const hilos = GmailApp.search('label:' + queryLabel_(CONFIG.ETIQUETA_PEDIR), 0, 1);
  if (!hilos.length) {
    Logger.log('No hay mails con la etiqueta «' + CONFIG.ETIQUETA_PEDIR +
               '». Ponésela a un mail en Gmail y volvé a probar.');
    return;
  }
  const msg = ultimoEntrante_(hilos[0]);
  if (!msg) { Logger.log('Ese hilo no tiene un mensaje entrante para responder.'); return; }
  Logger.log('── MAIL ENTRANTE ──\nDe: ' + msg.getFrom() + '\nAsunto: ' + msg.getSubject() +
             '\n\n' + recortar_(msg.getPlainBody(), 800));
  Logger.log('\n── BORRADOR QUE MIRROR DEJARÍA (prueba, NO creado) ──\n\n' + redactarBorrador_(msg));
}

/** CORRIDA REAL: solo los mails que VOS etiquetaste. Deja borradores. */
function mirrorRevisarMails() {
  const elPedir = obtenerEtiqueta_(CONFIG.ETIQUETA_PEDIR);
  const elListo = obtenerEtiqueta_(CONFIG.ETIQUETA_LISTO);
  const hilos = GmailApp.search('label:' + queryLabel_(CONFIG.ETIQUETA_PEDIR), 0, CONFIG.MAX_HILOS);
  let hechos = 0, saltados = 0;
  hilos.forEach(function (hilo) {
    try {
      const msg = ultimoEntrante_(hilo);
      if (!msg) { saltados++; return; }
      const texto = redactarBorrador_(msg);
      if (!texto) { saltados++; return; }
      hilo.createDraftReply(texto);   // ← SOLO borrador. Nunca envía.
      hilo.removeLabel(elPedir);      // sale de la cola: ya está hecho
      hilo.addLabel(elListo);
      hechos++;
    } catch (e) {
      Logger.log('Error en un hilo: ' + e);
    }
  });
  Logger.log('MIRROR dejó ' + hechos + ' borrador(es). Saltados: ' + saltados + '.');
}

/** Disparador automático (cada 15 min). Igual SOLO toca los mails que etiquetaste. */
function mirrorInstalarDisparador() {
  mirrorQuitarDisparador();
  ScriptApp.newTrigger('mirrorRevisarMails').timeBased().everyMinutes(15).create();
  Logger.log('✅ Listo: cada 15 min MIRROR revisa SOLO los mails con «' +
             CONFIG.ETIQUETA_PEDIR + '» y deja borradores.');
}

/** Quita el disparador automático. */
function mirrorQuitarDisparador() {
  ScriptApp.getProjectTriggers().forEach(function (t) {
    if (t.getHandlerFunction() === 'mirrorRevisarMails') ScriptApp.deleteTrigger(t);
  });
  Logger.log('Disparador detenido (si existía).');
}


// ─────────────────────────── internos ───────────────────────────

/** Llama a la IA a través del proxy del usuario (estilo OpenAI). Sin clave acá:
 *  la clave vive como secreto dentro del proxy. */
function llamarIA_(mensajes) {
  const url = CONFIG.PROXY_URL.replace(/\/+$/, '') + '/' + CONFIG.RUTA;
  const payload = {
    model: CONFIG.MODELO,
    messages: mensajes,
    temperature: CONFIG.TEMPERATURA,
    max_tokens: CONFIG.MAX_TOKENS
  };
  const res = UrlFetchApp.fetch(url, {
    method: 'post',
    contentType: 'application/json',
    headers: { 'Authorization': 'Bearer ' }, // el proxy inyecta la clave real
    payload: JSON.stringify(payload),
    muteHttpExceptions: true
  });
  const code = res.getResponseCode();
  const data = JSON.parse(res.getContentText() || '{}');
  if (code === 429) throw new Error('La cuota gratis del motor está ocupada (429). Reintentá luego o bajá MAX_HILOS.');
  if (data.error) throw new Error(data.error.message || JSON.stringify(data.error));
  if (code >= 400) throw new Error('El proxy respondió ' + code + '. Revisá PROXY_URL y RUTA.');
  const c = data.choices && data.choices[0] && data.choices[0].message && data.choices[0].message.content;
  return (c || '').replace(/<\|[^|]*\|>/g, '').replace(/<think>[\s\S]*?<\/think>/gi, '').trim();
}

/** Construye el prompt y devuelve el cuerpo del borrador. */
function redactarBorrador_(msg) {
  const sys =
    'Sos la extensión digital de tu usuario: redactás mails EN SU NOMBRE, con su voz y su estilo. ' +
    'No sos un asistente genérico — escribís como lo haría tu usuario.\n\n' +
    'QUIÉN SOS (perfil de tu usuario):\n' + CONFIG.PERFIL + '\n\n' +
    'TAREA: redactá un BORRADOR de respuesta al mail que te paso. Reglas:\n' +
    '- Idioma: ' + CONFIG.IDIOMA + ' (pero respetá el idioma del mail entrante si es otro).\n' +
    '- Tono y registro: los del perfil de arriba.\n' +
    '- HONESTIDAD: no inventes datos, fechas, cifras ni compromisos que no surjan del mail o del perfil. ' +
    'Si falta un dato, dejá un hueco entre corchetes, ej: [confirmar fecha], para que tu usuario lo complete.\n' +
    '- Devolvé SOLO el cuerpo del mail (sin asunto, sin "Para:", sin explicaciones tuyas).\n' +
    (CONFIG.FIRMA ? '- Cerrá con la firma: ' + CONFIG.FIRMA + '\n' : '');
  const entrante =
    'MAIL ENTRANTE\nDe: ' + msg.getFrom() + '\nAsunto: ' + msg.getSubject() + '\n\n' +
    recortar_(msg.getPlainBody(), CONFIG.MAX_CHARS_ENTRANTE);
  return llamarIA_([
    { role: 'system', content: sys },
    { role: 'user', content: entrante }
  ]);
}

/** Último mensaje ENTRANTE del hilo (no escrito por vos). */
function ultimoEntrante_(hilo) {
  const yo = (Session.getActiveUser().getEmail() || '').toLowerCase();
  const msgs = hilo.getMessages();
  for (let i = msgs.length - 1; i >= 0; i--) {
    const from = (msgs[i].getFrom() || '').toLowerCase();
    if (yo && from.indexOf(yo) !== -1) continue;
    return msgs[i];
  }
  return null;
}

function obtenerEtiqueta_(nombre) {
  return GmailApp.getUserLabelByName(nombre) || GmailApp.createLabel(nombre);
}
/** Formato del nombre de etiqueta para la búsqueda (espacios → guiones). */
function queryLabel_(nombre) {
  return String(nombre).replace(/\s+/g, '-');
}
function recortar_(s, max) {
  s = String(s || '').replace(/\r/g, '').trim();
  return s.length > max ? s.slice(0, max) + '\n…[recortado]' : s;
}

Honesto: para escribir el borrador, MIRROR le manda el texto del mail a tu proveedor de IA (el de tu proxy), que lo procesa según sus políticas. Por eso elegís vos, mail por mail, cuáles contestar — evitá etiquetar mails con datos sensibles. Cada borrador lo revisás y lo enviás vos.
